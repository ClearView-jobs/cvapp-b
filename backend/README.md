# Clear View Back End Code

## Setup 

### Getting Service Key
Go to the appropriate firebase project. Click the gear icon and go into project settings. 

Go to into the Service accounts tab and then click "Generate New Key" and download the key. Rename the file to ServiceAccountKey.json and put it in the config folder. `backend\config\`.


## Starting the Server

To start the project go into the backend directory using

`cd backend`

Then runt the script 

`npm run dev` 

to start the server.
